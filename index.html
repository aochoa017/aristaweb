Hosting para landings y webs estáticas

Ideal si tienes una web en HTML/CSS puro, Astro, Hugo u otro generador estático. Hosting rápido, seguro y sin complejidad.

  • Despliegue desde repositorio o ficheros
  • Certificado SSL incluido
  • Backup semanal
  • Dominio y DNS opcionales (consultoría)
🌐

Hosting WordPress básico

Para el autónomo que quiere tener su WordPress en marcha con las garantías mínimas: backup, seguridad y un servidor bien configurado.

  • PHP y MySQL optimizados
  • SSL + entorno seguro
  • Backup diario, retención 7 días
  • Actualizaciones bajo demanda
⚙️

Hosting WordPress gestionado

Me encargo de todo: actualizaciones de core, plugins y temas, backups extendidos, monitorización y prioridad en el soporte.

  • Todo lo del plan básico
  • Backup diario, retención 14–30 días
  • Actualizaciones planificadas incluidas
  • 2 restauraciones incluidas/año
  • Prioridad en respuesta
